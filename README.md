# MediBridge

Secure AI-assisted clinical document exchange.

## Project Structure

- `frontend/` — React application
- `backend/` — FastAPI REST API
- `workers/` — asynchronous event and AI workers
- `database/` — migrations, schemas, and seed data
- `infrastructure/` — AWS and Terraform configuration
- `contracts/` — shared API and event contracts
- `integration-tests/` — API and end-to-end testing
- `docs/` — architecture, security, and project documentation
- `mock-services/` — mocked external systems
